**사용자 여정 시나리오**
1. 로그인, 비밀번호 찾기, 회원가입
2. 랜딩페이지 접속, 시나리오 만들기 버튼으로 시나리오 생성 기능 진입
3. 시나리오 제목, 내용, 드롭 다운 메뉴를 통한 요소 선택
4. 시나리오 스토리 전개 방식 선택, 전개 강도 선택 
5. 4단계 스토리 생성(유저가 입력한 내용과 선택한 요소 그리고 스토리 전개방식 및 강도를 토대로 LLM이 디벨롭할 것)
6. 4단계 스토리 유저가 편집 or 디벨롭. 4단계에서 각 대표 썸네일(콘티) 생성
7. 유저는 4단계 스토리와 썸네일(콘티) 결정 후 다음 단계인 12단계 숏트 생성으로 진입
8. 12단계 숏트 생성은 4단계 스토리를 각 3개의 숏트로 총 12개 숏트로 LLM이 디벨롭하는 과정. 유저는 12단계 숏트를 받아 볼 수 있음. 여기서 기 제작된 4단계 썸네일(콘티)가 대표이미지로 설정돼있음. 콘티 다운로드 버튼 있어야함.
9. 12단계 숏트의 내용이 나열돼있고 각 내용에는 제목, 내용이 있음. 제목과 내용을 유저가 수정할 수 있음. 또한 12개 숏트에 할당된 콘티 공간이 있어, 유저가 콘티 생성 버튼을 통해 1개씩 숏트에 맞는 콘티를 생성할 수 있음. 만약 콘티가 마음에 들지 않을 시 내용 편집 후 재생성 버튼을 통해 재생성 진행. 각 콘티는 다운로드 버튼이 있음.
10. 모든 스토리와 콘티가 완성된 12단계 숏트는 가로 형태의 기획안으로 만들어져 다운로드 버튼을 통해 다운로드 할 수 있음. 
11. 여기서 만들어진 모든 스토리, 콘티는 콘텐츠 관리 탭에서 확인 할 수 있음 
12. 12개 숏트 페이지에서 프롬프트 생성 버튼을 통해 프롬프트 생성 페이지 이동 가능, GNB를 통해서도 이동 가능. 
13. 프롬프트 생성은 기 제작된 12개 숏트의 스토리와 콘티 이미지를 토대로 제작
14. 기 제작된 12개 숏트 중 생성을 원하는 숏트를 선택하여 생성버튼을 누르면 프롬프트가 생성됨
15. 영상 생성 버튼을 통해 AI 영상 생성 페이지로 이동. 
16. AI 영상 기능은 기 제작된 프롬프트와 콘티 이미지를 토대로 생성 진행
17. 유저는 AI 영상 기능 페이지에서 생성되는 영상을 로딩바를 보며 기다린 뒤 생성되면 영상 플레이어를 통해 확인 후 피드백 버튼을 통해 피드백을 남기거나 재생성 버튼을 통해 재생성 진행. 제작된 영상 파일은 콘텐츠 관리에서 확인 가능
18. 영상 피드백 페이지에서는 영상을 v1, v2, v3 슬롯을 활용하여 영상을 업로드 함.(supabase database에 저장, 300mb 이내) 
19. 영상 업로드 후 링크를 다른 유저에게 전송하여 피드백에 참여 시킴(게스트, 회원)
20. 참여한 다른 유저는 영상을 선택하여 '타임코드를 기반한 시점 피드백' 댓글 형태로 남길 수 있으며 해당 프로젝트 참여자(유저, 게스트)는 감정표현을 남길 수 있음(좋아요, 싫어요, 모호해요)
21. 피드백 받는 영상의 스크린샷 생성 및 다운로드, URL 공유, 영상 교체, 영상 삭제 등 보조 기능들이 수반됨.
22. 유저는 데이터 관리 페이지에서 생성된 스토리 텍스트, 이미지 콘티, 영상, 피드백 등을 대쉬보드와 함께 관리(수정, 삭제, 다운로드)할 수 있음.
