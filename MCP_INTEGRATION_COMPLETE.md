```bash
npm run test:mcp                # MCP 서버 상태 확인
npm run test:mcp:enhanced       # 기본 MCP 테스트
npm run test:mcp:integration    # 실제 연동 테스트
npm run test:mcp:website        # 웹사이트 테스트
npm run test:mcp:performance    # 성능 테스트
npm run test:mcp:ci             # 모든 MCP 테스트
```

### **모니터링 및 리포팅**
```bash
npm run monitor:mcp             # 실시간 모니터링 시작
npm run report:mcp              # 일일 성과 리포트
npm run notify:mcp:test         # 알림 시스템 테스트
npm run setup:mcp:notifications # 알림 설정
```
